Configure the TI1 as Input.

Parameters
TIM2_ICPolarityThe Input Polarity. This parameter can be one of the following values:
  • TIM2_ICPOLARITY_FALLING
  • TIM2_ICPOLARITY_RISING
TIM2_ICSelectionspecifies the input to be used. This parameter can be one of the following values:
  • TIM2_ICSELECTION_DIRECTTI: TIM2 Input 1 is selected to be connected to IC1.
  • TIM2_ICSELECTION_INDIRECTTI: TIM2 Input 1 is selected to be connected to IC2.
TIM2_ICFilterSpecifies the Input Capture Filter. This parameter must be a value between 0x00 and 0x0F.
Return values
None
