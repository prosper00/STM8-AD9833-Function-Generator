 stm8s_eval_spi_sd.cThis file provides a set of functions needed to manage the SPI SD Card memory mounted on STM8xx-EVAL board (refer to stm8s_eval.h to know about the boards supporting this memory). It implements a high level communication layer for read and write from/to this memory. The needed STM8 hardware resources (SPI and GPIO) are defined in stm8xx_eval.h file, and the initialization is performed in SD_LowLevel_Init() function declared in stm8xx_eval.c file. You can easily tailor this driver to any other development board, by just adapting the defines for hardware resources and SD_LowLevel_Init() function
